Glossary of Dry-Stone Walling

This glossary is missing its Plain-English Index. You can help the wiki by reading the Plain-English Index and tidying the entries against it.
Cross-section of a double-wall: the footing wide, the batter drawing the courses in, a throughstone across the heart, the coping on top.

These are the words a waller uses out in the field, set down plain so you can carry them to the stone. Each one is given just as it bears on the work — the way you'd be told it standing over a fresh heap with a string run between the frames. Start anywhere: the Contents below, the Random entry in the margin, or just take a word and follow it into its own entry and keep going.

Contents

    batter noun, verb

    The way you set a wall in as you course it up: each course of face-stone is set in from the course beneath, from the footing up to the coping. You set the batter to the frame and the line.

    bear verb

    What one stone is to another. You set a stone so its bed is borne by the stone beneath, and so it bears the stone set on it. Every stone in a wall both bears and is borne.

    bed noun

    What you bear a stone down on. Its bed is borne by the stone beneath, and you set every stone on its bed, so it is not pinned up on a joint.

    cheek-end noun

    The wallhead of a double-wall, where each course is borne on a throughstone and set back to the frame. You cope a cheek-end so a wall bears to its wallhead and courses no stone beyond it.

    cock-and-hen noun

    Coping you set in two: one stone up, one stone down, on and on across a wallhead. Each stone is borne by the line and pinned, so no stone is set off the frame.

    coping noun, verb

    The course of stone you set on a wall, so it bears down on every course beneath and no course is set on it. Cock-and-hen is one coping; you cope a wallhead so a wall is borne down and set.

    course noun, verb

    Stone you set across a wall, face-stone to face-stone with hearting between, all set to one bed. You course a wall up, course on course, from the footing to the coping.

    cripple-hole noun

    A smoot you set into a wall from the footing up two course or three, borne on a throughstone, so a wall is coursed over it and bears on.

    double-wall noun, verb

    Two wall of face-stone set to one batter, with hearting set between and a throughstone across each course or two. Every wall you course is a double-wall.

    face-stone noun

    The stone you set to the line, to each wall of a double-wall. You bear each face-stone on the face-stone beneath, pin it, and heart behind it.

    fillet noun

    A stone you wedge into a joint to bear a face-stone up to the line, where a course is not at one bed. It is a pinning you set in bed, not up.

    footing noun

    The course every other course is borne by. You set it down so a wall bears on it and on no course beneath, and you course up from the footing to the coping.

    frame, the noun

    The batter you set up at each cheek-end of a wall. You set the line between two frames, and course each face-stone to it, so a wall is set to one batter from footing to coping.

    hearting noun, verb

    The stone you set between two face-stone of a double-wall, so each face-stone is borne from behind and no face-stone is set up on a joint. You heart each course as you course it up.

    joint noun

    Where two stone of a course bear on each other. You set each face-stone so a joint is borne by the stone over it, not coursed up joint on joint, and you pin each joint set to the line.

    line, the noun

    What you set each face-stone to. It is set between the frame at each cheek-end, and you course up to it, so each face-stone of a wall is set to one batter.

    pinning noun, verb

    The stone you wedge into a joint to bear a face-stone up to the line, so it is not set off the frame. You pin every face-stone and heart behind it.

    raking noun, verb

    The batter you set back into a cheek-end, course by course down to the footing, so a wallhead is borne and coped in to a wall.

    set verb

    To bear a stone on its bed, to the line and to the batter. You set each stone, pin it, heart behind it, and course up — so a stone set is a stone that bears.

    smoot noun

    A course you set up on two face-stone with a throughstone over, and a wall coursed over it: a way set into a wall. A cripple-hole is a smoot; a water-shoot is a smoot.

    stone noun

    What every wall is set of. You bear each stone on the stone beneath, so it bears the stone over it; and a wall is stone on stone — face-stone, throughstone, hearting, coping — borne by stone and no stone beyond.

    throughstone noun

    The stone you set across a double-wall to bear on both face-stone at one course, so the two wall are borne as one wall. You course a throughstone in every two or three course up.

    wall noun

    Stone set on stone to bear by itself: a double-wall of two face-stone with hearting between, coursed up from a footing to a coping, set to a batter on the line, borne at each wallhead by a cheek-end. Every wall is stone that bears stone.

    wallhead noun

    Where a wall is set to a cheek-end and courses no stone beyond. You rake a wallhead back, bear each course on a throughstone, and cope it, so a wall bears to its wallhead and no course is set beyond.

    water-shoot noun

    A smoot you set into the footing, one course up, borne on a throughstone, so a wall is coursed over it and bears on.

    wedge noun, verb

    To pin a stone into a joint, so a face-stone is borne up to the line. You wedge a fillet or a pinning in where a course is not at one bed.
